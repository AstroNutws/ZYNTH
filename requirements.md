# Zynth Requirements

## Runtime

- Java 17 or newer
- Maven 3.9 or newer
- Windows 10/11, macOS, or Linux

## Database (optional for live connect/import)

- PostgreSQL-compatible database
- Supabase PostgreSQL supported
- Network access to DB host/port
- Valid DB credentials

## Build Dependencies

- JavaFX controls/graphics
- PostgreSQL JDBC driver
- Jackson databind
- JUnit 5

## Functional Requirements

- Visual table/column explorer grouped by schema
- Schema filter that limits explorer/canvas/code output
- Drag and move tables on canvas
- Non-overlapping auto layout
- Relationship creation and visualization
- SQL/Prisma generation
- Migration diff preview
- Undo/redo support
- Save/load `.zynth` project files
- Optional DB connect/import
- Optional apply schema SQL to connected DB

## Release Artifacts

- Runnable JAR
- Windows EXE installer/app bundle via `jpackage`
- Docs: `README.md` and `tutorial.md`
